# Public Material Anonymization Guide

Public project material must not include:

- broadcaster name;
- country or legal entity details;
- internal hostnames;
- IP addresses;
- passwords, tokens, SNMP communities, SIP credentials;
- exact studio identifiers;
- actual routing names;
- customer or vendor contractual details;
- unredacted configuration exports;
- production network topology.

Use neutral wording:

- “commercial broadcast environment”
- “professional radio infrastructure”
- “multi-vendor broadcast plant”
- “production AoIP network”
- “anonymized station evidence”

Replace real examples with structurally equivalent fictional examples.
