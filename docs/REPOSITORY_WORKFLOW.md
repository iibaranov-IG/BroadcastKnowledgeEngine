# Repository Workflow

## Document workflow

1. Draft in markdown.
2. Technical director reviews strategic and operational accuracy.
3. Validation engineer reviews shift-level realism.
4. ChatGPT revises structure and language.
5. Codex maintains consistency checks and repository structure.
6. Public version is anonymized and redacted.

## Release levels

- `internal-draft`: may refer to real architecture under controlled access.
- `partner-redacted`: safe for selected external review.
- `public`: no production identifiers or proprietary details.
